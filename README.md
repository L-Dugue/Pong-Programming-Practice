# PONG - PROGRAMMING PRACTICE - BUILD 1.0.0

## Project Description
Repository for a creation of Pong, made for practicing programming in C# and development in Unity.

**Features**
* Custom physics system
* Slightly random angles when rebounding off walls
* Scoreboard
* SFX
* Two player support

**Future Features**
* None

**Known Bugs**

* Weird physics interaction when ball hits behind the player.

## How to play.

**Without using an EXE** - Supported and recommended
1. Clone the repository
2. Navigate to UnityHub
3. Open up the cloned repository in UnityHub
4. Double-click the project in UnityHub
5. Wait for Unity to load, and start the game within Unity
6. ENJOY!

**With using an EXE** - Not Supported As Of Yet
1. Clone the repository
2. Navigate to the repository in your file explorer
3. Enter repository, and go to the folder "Builds"
4. Open up the folder with the newest build of the game
5. Double-click the EXE
6. ENJOY!
